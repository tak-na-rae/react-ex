Styled components

1. CSS 서식 구현 방법 비교

예제 #1

App.jsx

import React, { useState }  from 'react'
import './App.css'

function App() {
	const [color, setColor] = useState("yellow");
	const divClassName = "App " + color;

	return (
		<div className={divClassName}>
			<h1>Color</h1>
			<select onChange={e=>setColor(e.target.value)} value={color}>
				<option>yellow</option>
				<option>green</option>
				<option>blue</option>
				<option>red</option>
			</select>
		</div>
	);
}

export default App;

App.css

최초 렌더링

const [color, setColor] = useState("yellow");

const divClassName = "App " + color;

처음 렌더링 될 때, color 값은 “yellow” 이다.

따라서 divClassName에 대입되는 값은 “App yellow” 이다.

<div className={divClassName}>

이 Virtual DOM에 의해서 다음과 같은 html 태그가 출력된다.

<div class=”App yellow”>

이 html 태그에는 div.App {…} 서식과 div.yellow {…} 서식이 적용된다.

<select onChange={e=>setColor(e.target.value)} value={color}>
<option>yellow</option>
<option>green</option>
<option>blue</option>
<option>red</option>
</select>

이 select 태그에서 green 항목을 선택하면,

setColor(e.target.value) 함수가 호출되어 color 리액트 상태에 “green” 문자열이 대입된다.

리액트 상태 값이 변경되었으므로, App 컴포넌트가 다시 렌더링되어야 한다.

두 번째 렌더링

리액트 상태가 변경되어 다시 렌더링 되어야 하므로, App 함수가 다시 호출된다.

App 함수 본문의 코드들이 실행된다.

const [color, setColor] = useState(”yellow”);

const divClassName = "App " + color;

color 리액트 상태값이 “green” 이므로, divClassName에 “App green” 문자열이 대입된다.

<div className={divClassName}>

이 Virtual DOM에 의해서 다음과 같은 html 태그가 출력된다.

<div class=”App green”>

이 html 태그에는 div.App {…} 서식과 div.green {…} 서식이 적용된다.

예제 #2

App.jsx

import React, { useState }  from 'react'
import './App.css'

function App() {
	const [color, setColor] = useState("#ffa");

	return (
		<div className="App" style={{backgroundColor: color}}>
			<h1>Color</h1>
			<select onChange={e=>setColor(e.target.value)} value={color}>
				<option value="#ffa">yellow</option>
				<option value="#dfd">green</option>
				<option value="#ccf">blue</option>
				<option value="#fdd">red</option>
			</select>
		</div>
	);
}

export default App;

select 태그의 선택된 항목이 변경되면,

color 상태의 값은 “#ffa”, “#dfd”, “#ccf”, “#fdd” 중의 하나가 된다.

<div className="App" style={{backgroundColor: color}}>

노란색으로 칠한 부분, 즉 style props는 인라인 서식{inline style}이다.

style props의 값은 javascript 객체이어야 한다. 예: {backgroundColor: color}

props의 값이 문자열이면 따옴표로 묶지만 예: value=”hello”

props의 값이 문자열이 아니고 javascript 표현식이면, 중괄호로 묶어야 한다. 예: value={true}

style props의 값이 javascript 객체니까, 예: {backgroundColor : color}

중괄호로 묶어야 한다. 예: style={{backgroundColor : color}}

style props로 구현된 인라인 서식에서, CSS 서식명은 camel case로 변경되어야 한다.

CSS 서식명style props 서식명
background-colorbackgroundColor
font-sizefontSize
padding-leftpaddingLeft

그리고 style props로 구현된 인라인 서식에서, CSS 서식의 값은 javascript 표현식이어야 한다.

예

//표준 CSS
div#App {
	width: 100px;
	font-size: 10px;
	color: blue;
	background-color:#ffa;
}

<div id="App">

//style props
const bgcolor = "#ffa";

<div id="App" style={{
	width: "100px",
	fontSize: "10px",
	color: "blue",
	backgroundColor: bgcolor
}}>

2. styled-components

별도의 .css 파일에 서식을 구현하는 것이 약간 번거롭다.

그리고 style props의 값은 javascript 객체 문법이어서, CSS 문법과 달라 조금 헷갈릴 수 있다.

styled-components 라이브러리를 사용하면, tsx 파일에 css 서식을 구현할 수 있고,

표준 CSS 문법을 사용할 수 있고, 상태 변수나 props 변수를 사용할 수도 있다.

라이브러리 설치

npm install styled-components @types/styled-components

프로젝트 디렉토리에서 위 명령을 실행하자.

예제 #1

App.jsx

import React, { useState }  from 'react'
import styled from 'styled-components';

function App() {
	const [color, setColor] = useState("#ffa");
	const Div = styled.div`
		border: 1px solid gray;
		box-shadow: 5px 5px 5px gray;
		padding: 40px;
		margin: 20px;
		display: inline-block;
		background-color:${color};
	`;

	const H1 = styled.h1`
		margin: 10px;
	`;

	const Select = styled.select`
		padding: 4px;
		width: 150px;
		font-size: 12px;
	`;

	return (
		<Div>
			<H1>Color</H1>
			<Select onChange={e=>setColor(e.target.value)} value={color}>
				<option value="#ffa">yellow</option>
				<option value="#dfd">green</option>
				<option value="#ccf">blue</option>
				<option value="#fdd">red</option>
			</Select>
		</Div>
	);
}

export default App;

import styled from ‘styled-components’;

라이브러리 import

const Div = styled.div`
border: 1px solid gray;
box-shadow: 5px 5px 5px gray;
padding: 40px;
margin: 20px;
display: inline-block;
background-color:${color};

`;

역따옴표 문자 안에 css속성을 입력한다

위 코드에 의해, div 태그에 CSS 서식이 적용되어 Div 태그 컴포넌트가 생성된다.

${expression} 부분은 expression(표현식)의 값으로 교체된다.

즉 color 변수의 값이 ‘#ffe’ 이므로, background-color: ‘#ffe’; 가 된다.

예제를 좀 더 간결하게 구현할 수 있다.

예제 #2

App.jsx

import React, { useState }  from 'react'
import styled from 'styled-components';

function App() {
	const [color, setColor] = useState("#ffa");
	
	const Div = styled.div`
		border: 1px solid gray;
		box-shadow: 5px 5px 5px gray;
		padding: 40px;
		margin: 20px;
		display: inline-block;
		background-color:${color};
		& h1 { margin: 10px; }
		& select { padding:4px; width:150px; font-size: 12px; }
	`;

	return (
		<Div>
			<h1>Color</h1>
			<select onChange={e=>setColor(e.target.value)} value={color}>
				<option value="#ffa">yellow</option>
				<option value="#dfd">green</option>
				<option value="#ccf">blue</option>
				<option value="#fdd">red</option>
			</select>
		</Div>
	);
}

export default App;

const Div = styled.div`
border: 1px solid gray;
box-shadow: 5px 5px 5px gray;
padding: 40px;
margin: 20px;
display: inline-block;
background-color:${color};

 & h1 { margin: 10px; }
& select { padding:4px; width:150px; font-size: 12px; }

`;

Div 내부에 있는 & h1 태그, & select 태그에 서식을 적용한다.

이 부분은 표준 CSS 문법이 아니고 SCSS 확장 문법이다.

const Div 변수를 App 함수 지역 변수가 아니고, 전역 변수로 구현할 수도 있지만,

${color} 부분은 App 함수 밖으로 나갈 수 없다. color는 App 함수의 지역 변수이기 때문이다.

& h1 { margin: 10px; }

여기서 & 부분은 이 서식이 들어있는 div 태그를 가르킨다.

즉 이 서식은 아래와 같다.

div h1 { margin: 10px; }

예제를 좀 더 정리해서 구현할 수 있다.

예제 #3

App.jsx

import React, { useState }  from 'react'
import styled from 'styled-components';

const Div = styled.div`
	border: 1px solid gray;
	box-shadow: 5px 5px 5px gray;
	padding: 40px;
	margin: 20px;
	display: inline-block;
	background-color:${props => props.color};
	& h1 { margin: 10px; }
	& select { padding:4px; width:150px; font-size: 12px; }
`;

function App() {
	const [color, setColor] = useState("#ffa");
	
	return (
		<Div color={color}>
			<h1>Color</h1>
			<select onChange={e=>setColor(e.target.value)} value={color}>
				<option value="#ffa">yellow</option>
				<option value="#dfd">green</option>
				<option value="#ccf">blue</option>
				<option value="#fdd">red</option>
			</select>
		</Div>
	);
}

export default App;

const Div = styled.div`
border: 1px solid gray;
box-shadow: 5px 5px 5px gray;
padding: 40px;
margin: 20px;
display: inline-block;
background-color:${
props => props.color};
& h1 { margin: 10px; }

`;

이 코드는 서식이 적용된 div 컴포넌트를 생성하여 전역 변수 Div에 대입한다.

CSS 서식이 적용된 div 태그를 전역 변수로 구현 하였다.

이제 함수형 컴포넌트 본문에는 실행 로직과 태그 부분만 들어 있고

CSS 서식 부분은 전역 변수로 분리하였기 때문에, 함수형 컴포넌트 본문의 코드를 읽고 수정하기에 편하다.

background-color:${props => props.color};

이 부분은 background-color 서식에 Div 컴포넌트의 color props 값을 지정하는 코드이다.

즉 이 Div 컴포넌트를 사용할 때, color props 값이 전달되어야 한다.
